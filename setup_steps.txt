1. mkdir <proj_dir>
2. npm install --save react react-dom @types/react @types/react-dom
3. npm install --save-dev typescript typings tslint or npm install --dev typescript typings tslint
4. npm install --save-dev webpack webpack-dev-server awesome-typescript-loader
5. ./node_modules/.bin/tsc --init
6. Configure the package.json, tsconfig.json, webpack.config.js according to the requirements and follow as done in this project
7. npm run <build>



NB: For webpack version > 4, webpack watch mode not running properly here.
